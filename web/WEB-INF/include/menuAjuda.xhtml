                2 - O critério de permissão e habilitando usuário;

                               

               Administrador


                Este usuário tem o controle total sobre os demais usuários, com permissões de alterar, cadastrar, editar e excluir usuários, assim como, determinar papéis de acesso, ativar contas e alterar senha.                                

               Diretor


                Ele tem permissão sobre a geração de relatórios.                

               Supervisor


                Este controla os recursos mais avançado do sistema verificando e avaliando o andamento de cada etapa dos eventos e estimando espaços e condições de armazenamento, mantendo os profissionais que serão acionados nas ações de armazenagem, indicando as posições e as movimentações permitindo alimentar com informações globais com futuras consultas de relatórios. Principais ações do sistema permitida para este usuário, CRUD - Create (Criação), Retrieve (Consulta), Update (Atualização) e Delete (Destruição).

1. Receber Pedidos;
2. Classificar os Produtos Perigosos;
3. Sondar disponibilidade espaços para ocupação de armazenagem;
4. Manter Funcionários;
5. Manter Fornecedor;
6. Manter EPI (Equipamento de Proteção Individual);
7. Manter Veículo para Movimentação de Carga;
8. Manipular, Armazenar e acondicionar em seus respectivos recipientes;
               

               Operador


                A este usuário é permitido somente visualizar as ações já cadastrada no sistema.
